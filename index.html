Functions in python

In Python, a **function** is a block of reusable code that performs a specific task. Functions help in breaking down a problem into smaller, manageable pieces and are key to writing modular, organized code.

Defining a Function:

A function in Python is defined using the `def` keyword followed by the function name, parentheses (which may include parameters), and a colon. The code block inside the function is indented.

  • Syntax:

    • def function_name(parameters):
    # Code block
    # Return something if needed

    def greet():
    print("Hello, World!")

    To call or invoke the function, you simply use its name followed by parentheses:

    greet() # Output: Hello, World!

    Function with Parameters:

    Functions can take **parameters** (also known as **arguments**) that allow you to pass data into the function to work with. Parameters are specified within the parentheses during the function definition.

    def greet(name):
    print(f"Hello, {name}!")

    Calling the function with an argument:

    greet("Alice") # Output: Hello, Alice!
    greet("Bob") # Output: Hello, Bob!

    Function with Return Value:

  • A function can return a value using the `return` statement. This allows the function to output a result that can be used elsewhere in the program.

    • def add(a, b):
    return a + b

    Calling the function and capturing the returned value:

    result = add(3, 4)
    print(result) # Output: 7

    Default Parameters:

  • You can provide **default values** for parameters in case no argument is passed when calling the function. This helps to make parameters optional.

    • def greet(name="Guest"):
    print(f"Hello, {name}!")

    Calling the function with and without an argument:

    greet("Alice") # Output: Hello, Alice!
    greet() # Output: Hello, Guest!

    Variable-Length Arguments (Arbitrary Arguments):

    Sometimes, you may not know how many arguments will be passed to the function. Python provides two ways to handle such cases:

  • `args`: Used to pass a variable number of non-keyword arguments (positional arguments).
  • `kwargs`: Used to pass a variable number of keyword arguments (arguments passed by name).

    • def print_numbers(*args):
    for num in args:
    print(num)
    Calling the function:

    print_numbers(1, 2, 3, 4, 5)

    Output:
    1 2 3 4 5

    def print_info(**kwargs):
    for key, value in kwargs.items():
    print(f"{key}: {value}")
    Calling the function:
    print_info(name="Alice", age=25, city="New York")

    Output:

    name: Alice age: 25 city: New York

    Lambda Functions (Anonymous Functions):

  • In Python, you can also define small, one-line functions without a name using the `lambda` keyword. These are called **lambda functions** or **anonymous functions**.
  • Syntax:

    • lambda arguments: expression

    square = lambda x: x * x print(square(4)) # Output: 16

  • Lambda functions are useful when you need a simple function for a short period of time, like for use with higher-order functions (e.g., `map()`, `filter()`).

    • Functions as First-Class Citizens:

  • In Python, functions are **first-class citizens**, meaning they can be passed around as arguments, returned from other functions, and assigned to variables.

    • def apply_function(func, x):
    return func(x)

    result = apply_function(lambda x: x * 2, 5)
    print(result) # Output: 10

    Recursion:

  • A function can call itself. This is known as **recursion**. Recursive functions must have a **base case** to stop the recursion, otherwise they would result in an infinite loop.

    • def factorial(n):
    if n == 0: # Base case
    return 1
    else:
    return n * factorial(n - 1)
    Calling the function:

    print(factorial(5)) # Output: 120

    Summary:

  • Defining Functions: Use the `def` keyword to define a function.
  • Parameters: Functions can accept parameters to process data.
  • Return Values: Functions can return values using the `return` keyword.
  • Default Parameters: You can set default values for function parameters.
  • Variable-Length Arguments: Use `*args` and `**kwargs` to handle a variable number of arguments.
  • Lambda Functions: Anonymous one-line functions created with the `lambda` keyword.
  • Recursion: A function that calls itself to solve problems.
